* init.org --- An Emacs Configuration

** Introduction

This repository contains my Emacs configuration which is built using [[https://github.com/akirak/emacs-twist][emacs-twist]].
I use home-manager to install the configuration.

Note: This configuration is currently built and tested only on Windows Subsystem for Linux (WSL).

Caution: this repository is still alpha version.
I sometime use =git rebase -i= to clean up the history. Therefore, you need to execute script below to reset your repository.

#+BEGIN_SRC sh
  git fetch origin main
  git reset --hard origin/main
#+END_SRC

*** Commands

In zsh, you need to use a backslash before the #.
More information on metacharacters can be found in =man zshmisc=, in the "QUOTING" section.

*** Tips for home-manager integration
The example of =flake.nix= is below:
#+BEGIN_SRC nix
  {
    inputs = {
      nixpkgs.url = "github:nixos/nixpkgs/nixos-unstable";
      home-manager = {
        url = "github:nix-community/home-manager";
        inputs.nixpkgs.follows = "nixpkgs";
      };
      emacs-config.url = "github:/fneril058/emacs-config"; # or "path:local_path_to_repo"
      inputs.nixpkgs.follows = "nixpkgs";
    };

    outputs = { nixpkgs, home-manager, emacs-config,... }:
    let
      system = "x86_64-linux";    # or other system
      pkgs = nixpkgs.legacyPackages.${system};
    in
      {
        homeConfigurations."your_user_name" = home-manager.lib.homeManagerConfiguration {
          inherit pkgs;
          modules = [
            emacs-config.homeModules.twist
          ];
        };
      };
  }
#+END_SRC

*** Digression

Initially, I was using rubikitch's auto-install
([[https://rubikitch.hatenadiary.org/entry/20091221/autoinstall][auto-install.elでEmacsLispパッケージを自動インストール→anything.el関連を一括インストール - http://rubikitch.com/に移転しました]]).

Soon after, I started using Emacs's package.el in conjunction with it.
It was included by default in Emacs-24.
While melp has many packages, I was dissatisfied with the fact that package.el doesn't allow you to revert to previous versions.

I found [[https://github.com/emacscollective/borg?tab=readme-ov-file][emacscollective/borg]] and started using it, but cloning the entire repository felt like overkill.

Then I found [[https://github.com/akirak/emacs-twist][emacs-twist]] and started using that.

#+begin_src emacs-lisp
  ;;; Profiler
  (require 'profiler)
  (profiler-start 'cpu)

  ;; enable debug
  ;; (setq debug-on-error  t
  ;;       init-file-debug t)

  ;; Track `load-file-name'
  ;; The result will be shown in the buffer named *setup-tracker*.
  (defvar setup-tracker--level 0)
  (defvar setup-tracker--parents nil)
  (defvar setup-tracker--times nil)

  (when load-file-name
    (push load-file-name setup-tracker--parents)
    (push (current-time) setup-tracker--times)
    (setq setup-tracker--level (1+ setup-tracker--level)))

  (add-variable-watcher
   'load-file-name
   (lambda (_ v &rest __)
     (cond ((equal v (car setup-tracker--parents))
            nil)
           ((equal v (cadr setup-tracker--parents))
            (setq setup-tracker--level (1- setup-tracker--level))
            (let* ((now (current-time))
                   (start (pop setup-tracker--times))
                   (elapsed (+ (* (- (nth 1 now) (nth 1 start)) 1000)
                               (/ (- (nth 2 now) (nth 2 start)) 1000))))
              (with-current-buffer (get-buffer-create "*setup-tracker*")
                (save-excursion
                  (goto-char (point-min))
                  (dotimes (_ setup-tracker--level) (insert "> "))
                  (insert
                   (file-name-nondirectory (pop setup-tracker--parents))
                   " (" (number-to-string elapsed) " msec)\n")))))
           (t
            (push v setup-tracker--parents)
            (push (current-time) setup-tracker--times)
            (setq setup-tracker--level (1+ setup-tracker--level))))))

#+end_src

** Setup.el

#+begin_quote
  [[https://codeberg.org/pkal/setup.el][setup.el]] provides a macro to ease repetitive configuration patterns in Emacs. This is done by providing context sensitive local macros, that expand to regular, pure EmacsLisp.
#+end_quote

#+begin_src emacs-lisp
  (eval-when-compile
    (require 'setup)
    (require 'cl-lib)

    ;; https://www.emacswiki.org/emacs/SetupEl#h5o-4
    (defmacro defsetup (name signature &rest body)
      "Shorthand for `setup-define'.
  NAME is the name of the local macro.  SIGNATURE is used as the
  argument list for FN.  If BODY starts with a string, use this as
  the value for :documentation.  Any following keywords are passed
  as OPTS to `setup-define'."
      (declare (debug defun))
      (let (opts)
        (when (stringp (car body))
          (setq opts (nconc (list :documentation (pop body))
                            opts)))
        (while (keywordp (car body))
          (let* ((prop (pop body))
                 (val `',(pop body)))
            (setq opts (nconc (list prop val) opts))))
        `(setup-define ,name
           (cl-function (lambda ,signature ,@body))
           ,@opts)))

    (defsetup :package (package)
              "Fake installation of PACKAGE."
              :repeatable t
              :shorthand cadr))

#+end_src
